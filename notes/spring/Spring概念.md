# Spring概念

- Spring框架概述
    - Spring框架是一个轻量级的、开放源代码的J2EE应用程序框架。
    - Spring有两个核心部分：IOC和Aop
        1. IOC：控制反转，把创建对象的过程交给Spring进行管理。
        2. Aop：面向切面，不修改源代码的情况下，进行功能的增加和增强。
    - Spring框架的相关的特点
        1. 方便解耦，简化开发。
        2. Aop编程支持
        3. 方便程序的测试
        4. 方便整合各种优秀框架
        5. 降低Java EE API的使用难度
        6. 方便进行事务操作
